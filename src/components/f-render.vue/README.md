# FRender 渲染器

## 示例
有时在模板中需要动态渲染一部分`vnode`的节点，比如别的组件的`slot`。常用的方法是把整个模板用`render`函数重写，或者局部再写个组件，这些方法的成本都较高。

`<f-render>`可以方便解决这类问题。

``` htm
<f-render :vnode="$slots.title"></f-render>
```

### Props/Attrs

| --------- | ---- | ------- | ------- | ----------- |
