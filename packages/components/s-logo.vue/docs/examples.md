### 基本用法

``` html
<u-linear-layout>
    <u-logo></u-logo>
    <u-logo color="inverse" href="/proto-ui/" target="_blank"></u-logo>
</u-linear-layout>
```

### 添加副标题

``` html
<u-linear-layout>
    <u-logo href="/proto-ui/">原型组件</u-logo>
    <u-logo color="inverse" href="https://vusion.github.io/icon-sets" target="_blank">图标库</u-logo>
</u-linear-layout>
```
